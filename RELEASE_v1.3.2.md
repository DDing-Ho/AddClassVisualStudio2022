# CppClassHereVsix v1.3.2

`CppClassHereVsix`는 Visual Studio 2022에서 C++ 클래스를 더 편하게 만들 수 있게 해주는 확장 프로그램입니다.

기본 `추가 -> 클래스` 기능은 솔루션 탐색기에서 폴더나 필터를 우클릭해도 실제 `.h`, `.cpp` 파일이 프로젝트 루트에 생성되는 경우가 많습니다. 이 확장은 그 불편함을 줄이기 위해 `Add C++ Class Here...` 메뉴를 추가하고, 사용자가 선택한 위치를 기준으로 파일을 만들고 프로젝트에 바로 추가해줍니다.

## 설치하면 무엇이 추가되나요?

확장을 설치하면 아래 위치에서 `Add C++ Class Here...` 메뉴를 사용할 수 있습니다.

- 솔루션 탐색기에서 프로젝트, 폴더, 필터를 우클릭했을 때의 `추가` 메뉴
- 상단 `도구` 메뉴

이 메뉴를 누르면 전용 클래스 생성 다이얼로그가 열리고, 그 자리에서 클래스 이름과 파일 이름 등을 입력해 바로 파일을 만들 수 있습니다.

## 어떻게 동작하나요?

### 실제 폴더를 선택한 경우

- 실제 디스크의 그 폴더에 `.h`, `.cpp` 파일을 생성합니다.
- 프로젝트에도 해당 폴더 아래에 바로 추가됩니다.

### VC 필터를 선택한 경우

- 실제 디스크에는 프로젝트 루트에 파일을 생성합니다.
- 프로젝트 안에서는 선택한 필터 아래에 추가됩니다.

즉, 필터를 선택했다고 해서 실제 디스크에 같은 이름의 폴더가 생기지는 않습니다.

### 프로젝트 루트를 선택한 경우

- 실제 디스크와 프로젝트 모두 프로젝트 루트 기준으로 파일을 추가합니다.

## 무엇을 입력할 수 있나요?

클래스 생성 다이얼로그에서 아래 항목을 입력할 수 있습니다.

- 클래스 이름
- `.h` 파일 이름
- `.cpp` 파일 이름
- 기본 클래스
- 액세스 지정자
- `인라인` 옵션

`인라인` 옵션을 체크하면 `.cpp` 없이 헤더 파일만 생성할 수 있습니다.

## 어떤 점이 편해지나요?

- 실제 폴더를 우클릭했을 때 그 폴더에 바로 파일을 만들 수 있습니다.
- 만든 파일이 자동으로 프로젝트에 추가됩니다.
- 헤더 파일을 자동으로 열어줍니다.
- Visual Studio UI 언어에 따라 한국어/영어로 표시됩니다.
- Visual Studio 테마 색을 따라갑니다.

## 릴리스 자산

- `Installer\CppClassHereVsix-1.3.2.vsix`

## 설치 방법

1. Visual Studio 2022를 종료합니다.
2. 이 릴리스에서 `CppClassHereVsix-1.3.2.vsix`를 다운로드합니다.
3. `.vsix` 설치 파일을 실행합니다.
4. 설치가 끝나면 Visual Studio를 다시 실행합니다.

## 지원 대상

- Visual Studio 2022
- `vcxproj` 기반 C++ 프로젝트

## 참고

- 빌드 없이 바로 설치할 수 있습니다.
- 소스를 직접 빌드하려면 Visual Studio 2022 SDK가 필요합니다.
- 지원 UI 언어는 한국어와 영어입니다.
